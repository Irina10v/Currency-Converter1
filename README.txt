Описание программы
Графическое приложение на Python для конвертации валют с использованием внешнего API. Приложение позволяет получать актуальные курсы валют, конвертировать суммы, сохранять историю операций в JSON и управлять ею.

Требования для запуска
Что нужно установить на компьютер:

Python 3.10 или выше
Библиотеки: pip install tkinter, pip install requests, pip install os
Как получить API-ключ
Перейдите на сайт ExchangeRate-API
Нажмите "Get Free API Key"
Зарегистрируйтесь (бесплатный тариф позволяет 1500 запросов в месяц)
После регистрации вы получите API ключ
Вставьте ключ в поле "API Key" в приложении и нажмите "Сохранить API Key"
Бесплатный тариф включает:

1500 запросов в месяц
Обновление курсов каждые 24 часа
Поддержка 160+ валют
Как запустить
git clone https://github.com/xyz-dsgn/Currency-Converter
cd Currency-Converter
python main.py
Как пользоваться
Конвертация валют

Выберите валюту "Из" (например, USD)
Выберите валюту "В" (например, EUR)
Введите сумму для конвертации (положительное число)
Нажмите "Конвертировать"
Результат отобразится в поле "Результат" и в истории
Пример работы
Запуск программы — открывается окно с настройками и историей.
Настройка API:
Ввести API ключ: your-api-key-here
Нажать Сохранить API Key
Нажать Обновить список валют
Конвертация валюты:
Из валюты: USD
В валюту: EUR
Сумма: 100
Нажать Конвертировать
Результат: 92.00 EUR
Просмотр истории:
В таблице появляется запись:
Дата: 2026-05-04 15:30:22
Сумма: 100.00
Из: USD
В: EUR
Результат: 92.00
Курс: 0.9200
Сохранение истории:
Нажать Сохранить историю — данные сохраняются в conversion_history.json
